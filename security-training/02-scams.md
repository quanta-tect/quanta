# 🎣 Course 02: Avoiding Crypto Scams

**Audience**: All users
**Duration**: 45 minutes

> Crypto loses **$2B+ per year** to scams. Most could be avoided with this knowledge.

## Module 1: The 12 most common scam patterns (20 min)

### 1. Fake support DM
> "Hey I'm support from QUANTA, saw your post about the failed tx, click here to fix"
- **Truth**: Real support NEVER DMs first
- **Defense**: Disable DMs from non-friends. If unsure, ask publicly.

### 2. Wallet drainer link
> "Claim your free 1000 QTA: quanta-airdrop.com"
- **Truth**: Sites ask you to "verify wallet" → unlimited approval → drain
- **Defense**: Only claim from official URLs you typed yourself

### 3. Phishing site (typo squat)
> `qunata.foundation` (vs real `quanta.foundation`)
- **Defense**: Bookmark official site. Type URL char-by-char.

### 4. Fake hardware wallet
> "Ledger Nano X — 50% off!" on eBay/Amazon third-party
- **Truth**: Pre-loaded with attacker's seed
- **Defense**: Buy ONLY direct from manufacturer

### 5. Pig butchering
> Romance app match → "let me show you my trading platform"
- **Truth**: Platform shows fake profits until you deposit big → "withdrawal blocked, pay tax to unlock"
- **Defense**: Anyone you've never met in person ≠ trusted

### 6. Honeypot token
> "BUY $PEPENEW! 100x in 2 days!"
- **Truth**: You can buy, no one can sell (sell function disabled)
- **Defense**: Check honeypot.is BEFORE buying any new token

### 7. Rug pull
> Team launches token, hypes, collects liquidity, deletes Discord, vanishes
- **Defense**: Avoid anon teams. Check liquidity locks. Check vesting.

### 8. Discord webhook scam
> Hacked Discord bot posts "official announcement" with malicious link
- **Defense**: Verify announcements come from KNOWN admin accounts (check profile age)

### 9. Twitter impersonation
> "Vitalik" tweets "send 1 ETH, get 10 back" with verified-looking checkmark
- **Truth**: Bought checkmark, fake account
- **Defense**: Verify follower count, account age, mutuals. If too good to be true, it is.

### 10. Job scam
> "Senior Solidity Developer needed, $300K, take this coding test"
- **Truth**: "Test" repo has malicious dependency that steals your keys
- **Defense**: Run test repos in Docker, never on machine with crypto wallets

### 11. Wrench attack
> Real-world coercion — kidnapping, "give us your seed phrase"
- **Defense**: Don't publicize crypto wealth, use duress signals (see Course 05)

### 12. Address poisoning
> Attacker sends 0 QTA from address looking IDENTICAL to your contact
- **Truth**: You later copy "your contact" from history, but it's attacker's address
- **Defense**: Verify FULL address before sending. Use address book, not history.

## Module 2: Recognize the patterns (10 min)

### Universal scam red flags

- ⚠️ Urgency ("act now or lose it!")
- ⚠️ Too good to be true (100x, free money)
- ⚠️ Pressure to keep secret ("don't tell support")
- ⚠️ Asking for seed phrase / private key
- ⚠️ Asking you to install unfamiliar software
- ⚠️ Promises of guaranteed returns
- ⚠️ Unknown people DMing you
- ⚠️ Misspelled URLs / contracts
- ⚠️ Requests to "verify wallet" by signing
- ⚠️ Pressure to act outside normal channels

If you see 2+ of these → 99% scam.

## Module 3: Verification techniques (10 min)

### Verify a URL is real
1. Check exact spelling
2. Check certificate (click padlock in browser)
3. Cross-reference with official Twitter bio
4. Check community Discord pinned messages

### Verify a contract is real
1. Get address from OFFICIAL site (bookmarked)
2. Check on Etherscan/Basescan
3. Verify "Verified Contract" badge
4. Cross-reference with project's GitHub deployment file
5. Check community can attest to address

### Verify a tweet/announcement is real
1. Check Twitter handle (not display name) — verify exact spelling
2. Check account creation date (recent = suspicious)
3. Check follower count (real projects have growing followers over years)
4. Look at replies — are people calling it scam?
5. Cross-reference with official Discord/blog

### Verify a person is real
1. Long-standing accounts across multiple platforms
2. Real-name verification (LinkedIn, etc.)
3. Public talks / podcast appearances
4. Other community members vouch
5. Video call before any deal

## Module 4: What to do if scammed (5 min)

### Immediate steps (first hour)
1. **Move remaining funds** to a new wallet from a clean computer
2. **Revoke ALL approvals** at revoke.cash
3. **Don't pay "recovery experts"** — they're round 2 of scam
4. **Document everything**: screenshots, addresses, conversations
5. **Report to community** — protects others

### Follow-up (first week)
1. Report to FBI IC3 (US): ic3.gov
2. Report to OFAC if attacker is in sanctioned region
3. Submit to Chainalysis Reactor (free)
4. Submit to ScamSniffer database
5. Notify exchanges (some can freeze if funds reach KYC'd account)

### Acceptance (first month)
- Most stolen funds are NOT recovered. Accept this early.
- Treat as $X tuition fee for permanent lesson
- Share story to help others avoid same mistake
- Don't blame yourself for too long — scammers are professionals

## TL;DR — The 3 unbreakable rules

1. **No legitimate party EVER asks for your seed phrase.** Ever. For any reason.
2. **If it's too good to be true, it is.** Free money = trap.
3. **When unsure, ask publicly.** DMs are scammer territory.

Stay safe out there. 🛡️
