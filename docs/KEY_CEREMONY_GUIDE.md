### BƯỚC 1 — INIT LEDGER (LẦN ĐẦU)

Mỗi Ledger làm 1 lần duy nhất:

1. Mở Ledger → **Setup as new device**
2. Ghi seed phrase lên steel plate — **KHÔNG** viết ra giấy thường
3. Chọn PIN code (4-8 chữ số)
4. Install apps: Ethereum, Bitcoin (backup)
5. Firmware update nếu có — check ledger.com/live
6. **Test:** Gửi 0.001 ETH vào Ledger → chuyển ra → xác nhận

**Ledger assignment:**
```
Ledger #1 — Operator — personal hot backup
Ledger #2 — Co-founder / technical advisor
Ledger #3 — Luật sű / accountant / trusted 3rd
Ledger #4 — Cold backup — két sắt ngân hàng
Ledger #5 — Emergency — người thân / sealed envelope
```

### BƯỚC 2 — FUND DEPLOYER WALLET

**Deployer = ephemeral — fresh address — KHÔNG dùng Ledger #1 cá nhân**

1. Tạo deployer address từ `~/.foundry/keystores/` — generated by Foundry
2. Nạp đúng **0.05 ETH Base mainnet** (gas deploy + key wipe)
3. **SAVE private key** encrypted — 1Password / VeraCrypt
4. **KHÔNG** dùng deployer này làm treasury / admin bất kỳ

### BƯỚC 3 — MULTISIG DEPLOY (GNOSIS SAFE)

**Deploy TRƯỚC token contract — BẮT BUỘC**

1. Mở app.safe.global → Base mainnet → Create new Safe
2. Owners: Ledger #1 + #2 + #3 + #4 + #5
3. Threshold: **3/5** (start 2/3 trong 90 ngày đầu nếu team nhỏ → upgrade 3/5 trước TVL >$50K)
4. Fund Safe: 0.001 ETH (test tx)
5. Test 3 transactions:
   - 0.001 ETH out → Ledger #1 → Ledger #2 → Ledger #3 sign
   - Confirm 2/3 threshold works
   - Rotate 1 signer → test recovery drill

**Safety:**
- **VN DRIVE — USB encrypted — VIDEO record整个过程** — bảo vệ khỏi key loss

## BƯỚC 4 — DEPLOY QTA V2 (SCRIPT + LEDGER)

**Deploy `QuantaVestingWallet` và `QuantaTreasuryController` TRƯỚC `QuantaTokenV2`**

### 4.1 — Deploy vesting + treasury + rewards
```bash
cd contracts/script
forge script DeployQTAV2.s.sol \
  --rpc-url $BASE_MAINNET_RPC \
  --private-key $DEPLOYER_KEY \
  --broadcast
```

**IN TREMINAL OUTPUT:**
```
QuantaVestingWallet: 0x...
QuantaTreasuryController: 0x...
QuantaRewardsDistributor: 0x...
QuantaTokenV2: 0x...
Total Supply: 1000000000000000000000000000000
```

### 4.2 — Verify on Basescan
```bash
forge verify-contract 0x... QuantaTokenV2 \
  --chain base --watch
# Repeat for vesting, treasury, rewards
```

### 4.3 — Post-deploy checks
```bash
cast call 0xQTA "totalSupply()" → 1000000000000000000000000000000 ? (YES)
cast call 0xQTA "hasRole(bytes32,0xtreasury)" MINTER_ROLE → MUST = false
cast call 0xQTA "owner()" → ? MUST = treasuryMultisig (NEVER deployer)
cast call 0xVesting "releasable()" → 0 at T0+364d
cast call 0xTreasury "timelock()" → 172800 (48h)
```

### 4.4 — Distribution transfers from treasury
```bash
# Approve vesting for 100M QTA
cast send $TREASURY_MULTISIG "approve(address,uint256)" $VESTING 100000000e18
# Fund vesting
cast send $VESTING "fund(uint256)" 100000000e18

# Approve rewards for 300M QTA
cast send $TREASURY_MULTISIG "approve(address,uint256)" $REWARDS 300000000e18
# Direct transfer (rewards uses emit(), give QTA balance directly)
cast send $TREASURY_MULTISIG "transfer(address,uint256)" $REWARDS 300000000e18
```
