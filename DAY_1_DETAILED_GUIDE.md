## STEP 1 — Create Logo + Branding Assets (60 min)

### 1.1 Logo design (30 min)

**Free option A — Recraft.ai (recommended)**

1. Open https://www.recraft.ai
2. Sign up free (10 credits/day)
3. Click "Create" → choose "Image"
4. Paste this exact prompt:

```
Minimalist logo design, atom symbol with electron orbits,
purple to cyan gradient (#7c3aed to #06b6d4),
dark background, vector style, futuristic crypto aesthetic,
clean lines, scalable, transparent background,
centered composition, professional brand identity, 1024x1024
```

5. Generate 4 variations, pick best
6. Download as PNG with transparent background

**Free option B — Hatchful by Shopify**

1. https://hatchful.shopify.com
2. Choose "Technology" → "AI" or "Crypto"
3. Browse templates, customize colors to purple/cyan
4. Download free pack

**Free option C — Bing Image Creator**

1. https://www.bing.com/create
2. Sign in with Microsoft account
3. Paste same prompt as Recraft
4. Get 4 images, download best

### 1.2 Generate variants (15 min)

You need 3 versions of the logo:

| File | Size | Use |
|------|------|-----|
| `logo.png` | 512×512 (transparent bg) | Twitter avatar, Discord icon |
| `logo-banner.png` | 1500×500 | Twitter banner |
| `favicon.ico` | 32×32 or 64×64 | Website tab icon |

**Tools to resize (free)**:

1. **Photopea** (https://www.photopea.com) — free Photoshop in browser
   - File → Open → upload your logo
   - Image → Image Size → set to 512×512
   - File → Export As → PNG (transparent)
   - Repeat for other sizes

2. **Favicon.io** (https://favicon.io)
   - Upload logo → auto-generates favicon

3. **Canva** (https://canva.com)
   - Custom size 1500×500
   - Drop logo, add tagline text, export

### 1.3 Create banner (15 min)

Twitter/X banner = 1500×500px.

Layout suggestion:
```
┌──────────────────────────────────────────────────────────┐
│                                                          │
│    [⚛ LOGO]    QUANTA                                    │
│                                                          │
│                Quantum-safe • AI-native • Open source    │
│                                                          │
│                github.com/your-repo                      │
└──────────────────────────────────────────────────────────┘
```

In Canva or Photopea:
- Background: dark gradient (#0a0a14 to #1a0a2e)
- Logo on left (200×200)

## STEP 5 — (Optional) Buy a Domain (30 min, $1-10)

Skip if you want to use GitHub Pages URL.

### Recommended registrars (cheapest first)

| Registrar | .xyz price | .com price | Pros |
|-----------|-----------|------------|------|
| **Spaceship.com** | $0.99-3 | $10-12 | Cheapest .xyz |
| **Cloudflare Registrar** | n/a | $10 (at cost) | No markup |
| **Porkbun** | $2-5 | $9-11 | Cheap renewals |
| **Namecheap** | $1-3 | $9-13 | Most popular |

**Avoid**: GoDaddy (predatory renewals), Wix (lock-in), .tk/.ml domains (blocked everywhere)

### Buy `.xyz` from Spaceship

1. Go to https://www.spaceship.com
2. Search: `quanta.xyz` or `quantacoin.xyz` or `quantaprotocol.xyz`
3. If available for $0.99-3, add to cart
4. Create account (free)
5. Pay via card or crypto
6. **Skip all upsells** (privacy protection is included free)
7. Total: should be < $5

### Point domain to GitHub Pages

After purchase:

1. In Spaceship dashboard → Domains → your domain → DNS
2. Delete any existing records
3. Add these A records (point to GitHub):
   ```
   Type: A   Host: @   Value: 185.199.108.153
   Type: A   Host: @   Value: 185.199.109.153
   Type: A   Host: @   Value: 185.199.110.153
   Type: A   Host: @   Value: 185.199.111.153
   ```
4. Add CNAME:
   ```
   Type: CNAME   Host: www   Value: YOUR_USERNAME.github.io
   ```
5. Save

In GitHub repo:
1. Settings → Pages
2. Custom domain: enter `quanta.xyz` (your domain)
3. Save → wait for DNS check
4. ✅ Enforce HTTPS

DNS propagation: 5 min - 24 hours.

Add CNAME file to repo:
```bash
echo "quanta.xyz" > CNAME  # your actual domain
git add CNAME
git commit -m "docs: add custom domain"
git push
```
